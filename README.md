This is second version of the Resource Biter mod. The original is here: https://github.com/pezplease/BiterResourceNests/.

Resource biters aims to add a fun twist to normal biters by adding specilized biters to the resource patches in the game. Iron, Copper, Coal, Stone and Uranium biters all have their own stats. 
- There is an option to allow resource biters to expend to new nests or only be on the resource.
- There is an option to remove all normal biters from the game.

This version aims to mesh a lot better with vanilla gameplay.

Basic Language Support for: English, Deutsch, Français, 日本語, Polski, Português Brasileiro, Русский, 简体中文
